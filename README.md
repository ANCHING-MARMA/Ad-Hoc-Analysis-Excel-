# 📊 Excel Sales Dashboard

## 📌 Project Overview

This project is an **interactive Excel Dashboard** built to analyze sales performance across different regions and time periods. The dashboard helps users quickly understand key business metrics through dynamic charts and filters.

## 🎯 Objectives

* Analyze regional sales performance
* Track sales trends over time
* Visualize category contribution to total sales
* Provide interactive filtering for better insights

## 📂 Dataset

The dataset contains sales-related information such as:

* Region
* Sales amount
* Product category
* Time period

## 📊 Dashboard Features

* **Interactive Slicers** for region filtering (Central, East, South, etc.)
* **Sales Trend Line Chart** to track performance over time
* **Category Distribution Pie Chart**
* **KPI Indicators** for quick performance understanding
* Dynamic charts that update automatically based on filters

## 🛠 Tools Used

* **Microsoft Excel**
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning and Formatting

## 📈 Key Insights

* Certain regions contribute significantly more to total sales.
* Sales trends show fluctuations across different periods.
* Product category distribution highlights major revenue drivers.
📊 Key Performance Indicators (KPIs)

1️. Total Sales

Shows the overall revenue generated in the selected period.
Helps understand the overall business performance.

2️. Total Orders / Transactions

Total number of sales transactions.
Useful to measure the sales activity level.

3️. Average Sales per Order

Average revenue generated per transaction.
Helps identify customer purchase behavior.

4️. Top Performing Regions

Identifies the region generating the highest sales.

5️. Sales Growth Trend

Shows how sales are increasing or decreasing over time.

6️. Category Contribution (%)

Shows which product category contributes the most to total sales.

7️. Regional Sales Distribution

Compares sales performance across different regions (Central, East, South, etc.).
